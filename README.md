Реализованный функционал:
✅ Интерактивная карта с метками НКО городов Росатома
✅ Система фильтрации (по городу, направлению деятельности, поиск по названию)
✅ Полноценные карточки организаций со всей требуемой информацией
✅ Возможность добавления новых НКО через веб-форму
✅ Адаптивный дизайн для мобильных устройств
✅ Фирменная цветовая схема Росатома

Зависимости проекта:
Flask==2.3.3
Flask-SQLAlchemy==3.0.5
geopy==2.3.0

Архитекстура
app.py              # Основной модуль Flask приложения
templates/          # HTML шаблоны
  base.html       # Базовый шаблон (навигация, футер)
  map.html        # Главная страница с картой
  organizations.html # Страница списка всех организаций
  create.html     # Форма добавления новой организации
static/
  css/main.css    # Стили проекта
  js/map.js       # Логика работы карты
